Es importante leer este archivo antes de intentar ejecutar la práctica, en otro
caso es probable que el proyecto tenga errores o no funcione correctamente.

El proyecto está montando en Netbeans para evitar comandos muy largos en la terminal
y se basa en una biblioteca de Java llamada Graphstream para su principal
funcionamiento y la interfaz gráfica que produce, esta puede ser revisada y/o
descargada desde el siguiente enlace:

http://graphstream-project.org/

Por esta misma razón existe un problema, la práctica depende de dos .jars obtenidos 
de Graphstream (gs-algo-1.3.jar y gs-core-1.3.jar) y debido a las políticas de Google
No es posible enviar el proyecto con estos archivos, así que será necesario descargar
las librerías de Graphstream, y agregar los .jar como librerias en /dist/lib o en 
cualquier otro directorio del proyecto para que este funcione correctamente.

En otro caso se podrá descargar la práctica completamente funcional desde Dropbox en 
el siguiente enlace (es probable que haya mas .jar en algun otro directorio del proyecto,
asi que descargarlo tambien seria una buena opcion):

-- El enlace va aqui --

Una vez esto sea realizado, bastará con correrlo y colocar el archivo que se quiera leer 
en el directorio principal del proyecto, es decir el mismo donde está este archivo de texto
y una vez que corra el programa se podrá introducir el nombre de este en la entrada estándar 
de Netbeans para que corra el algoritmo y muestre la gráfica.
