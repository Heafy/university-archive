Una vez compilado el archivo Search.java se ejecuta como se especifica en la
práctica 3:

java Search file.txt elemento nombreAlgoritmo 

Donde file.txt es el archivo de texto que contiene los elementos del espacio de 
búsqueda(Números separados por ' ').
Donde elemento es el numero a buscar.
Donde nombreAlgoritmos es:
- 'secuencial'
- 'binaria'
- 'exponencial'
- 'interpolacion'
Para elegir el algoritmo con el que se quiere buscar.

No se anexan ejemplos para su uso.

Adicionalmente existe un metodo para generar listas ordenadas, ese se ejecuta con el
siguiente comando:

java Search aleatorio

Y mostrara un arreglo aleatorio ordenado, la explicacion de este algoritmo está en la 
documentación del código.
