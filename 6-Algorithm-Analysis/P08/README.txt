Una vez compilado el todos los archivos javac *.java se ejecuta como se especifica en la
práctica 8:

java CoberturaVertices file

Donde file es el archivo de texto que contiene la información de la grafica de la siguiente
forma:
-En la primer linea, los vértices de la grafica separados por ’,’.
-De la segunda linea en adelante, pares de vertices separados por ’,’ que
indican las aristas de la gráfica, ademas de su peso.
