El archivo Ordenamiento.java contiene los algoritmos Bubble Sort y Exchange Sort
Una vez compilado el archivo este se ejecuta de una manera similar a la práctica 3:

java Ordenamientos file.txt nombreAlgoritmo ascendente/descendente

Donde file.txt es el archivo de texto que contiene los elementos del espacio de 
búsqueda(Números separados por ' ').
Donde nombreAlgoritmos es 'exchange' o 'bubble' La palabra sort no es necesaria.
Y donde ascendente/descendente es la manera en la que quiere que se ordene el algoritmo,
es necesario la palabra completa.

Se anexan un directorio con ejemplos de la práctica 3 más un par agregados por mi para las pruebas
